libclc-20.1.1

Introduction to libclc

The libclc package contains library requirements of the OpenCL C programming language (provides header files but no libraries itself).

  • Download (HTTP): https://github.com/llvm/llvm-project/releases/download/llvmorg-20.1.1/libclc-20.1.1.src.tar.xz

libclc Dependencies

Required

LLVM-20.1.1 (with Clang) and SPIRV-LLVM-Translator-20.1.0

Installation of libclc

Install libclc by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=/usr \
      -D CMAKE_BUILD_TYPE=Release  \
      -G Ninja .. &&

ninja

To test the results, issue: ninja test.

Now, as the root user: 

ninja install

Command Explanations

Note

Inspect CMakeLists.txt and other CMake related files for a full list of options.
