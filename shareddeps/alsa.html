ALSA

The Linux kernel now provides ALSA support by default. However, applications need to interface to that capability. The following three sections of the book deal with the separate components of ALSA. More Alsa packages if needed have installation instructions at https://www.linuxfromscratch.org/blfs/view/svn/multimedia/libdriv.html.
